# Project python BTL - Web quản lí bán quần áo

Web bao gồm các tính năng cần thiết như đăng nhập/ đăng ký tài khoản, giỏ hàng và đơn hàng. Ngoài ra, web còn có tích hợp tính năng quản lí cửa hàng để dễ dàng chỉnh sửa cơ sở dữ liệu. Web dựng từ framework flask, kết hợp html/css/js với python và mysql.
